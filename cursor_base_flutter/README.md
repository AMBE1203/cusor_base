# 🚀 Flutter AI Development Workflow (Cursor + Memory Bank)

This project integrates two powerful approaches to build a structured and scalable Flutter development workflow using AI (Cursor):

- Cursor Memory Workflow from https://github.com/vanzan01/cursor-memory-bank  
- Flutter coding standards from https://github.com/Wreos/flutter-cursor-plugin  

👉 Goal: Ensure AI-generated code is **consistent, maintainable, and follows a clear engineering process**

---

# 🧠 How It Works

## 1. Memory-driven Workflow (WHAT & WHEN)

Using cursor-memory-bank to:
- Manage task lifecycle
- Control AI thinking phases
- Maintain context across steps

Workflow commands:

| Command | Purpose |
|--------|--------|
| `/van` | Analyze request & create task |
| `/plan` | Generate implementation plan |
| `/creative` | Explore design decisions |
| `/build` | Implement code |
| `/reflect` | Review and validate |
| `/archive` | Finalize and store knowledge |
